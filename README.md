# XeuzCloud - Google Drive Clone

A high-fidelity Google Drive clone built with Next.js (App Router) and Material UI.

## Features

- **Google Drive UI**: Authentic Material Design implementation.
- **File Browser**: Toggle between Grid and List views.
- **File Upload**: Drag and drop support using `react-dropzone`.
- **Upload Progress**: Real-time progress notifications.
- **Responsive Design**: Mobile-friendly sidebar and layout.
- **Theming**: Custom Google Drive theme using MUI.

## Tech Stack

- **Framework**: Next.js 16
- **UI Library**: Material UI (MUI) v6
- **Icons**: MUI Icons Material
- **Drag & Drop**: React Dropzone

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `src/components`: UI components (DriveLayout, DriveMain, DriveSidebar, DriveHeader).
- `src/theme`: Client-side theme configuration.
- `src/data`: Mock data for file browser.
